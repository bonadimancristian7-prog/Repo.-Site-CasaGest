# CasaGest

Site estático de gestão residencial, feito com HTML, CSS e JavaScript puros.

## Arquivos

- `index.html`: estrutura e conteúdo do site.
- `style.css`: visual, tema claro/escuro e responsividade.
- `script.js`: navegação, CEP, contas, moradores e salvamento local.

## Como abrir

Abra o arquivo `index.html` no navegador. Para a consulta de CEP funcionar de
forma consistente, use uma extensão de servidor local ou publique no GitHub
Pages.

## Como enviar para o GitHub

1. Crie um repositório no GitHub.
2. Envie os quatro arquivos desta pasta para a raiz do repositório.
3. No repositório, abra **Settings → Pages**.
4. Em **Build and deployment**, escolha **Deploy from a branch**.
5. Selecione a branch `main`, a pasta `/ (root)` e clique em **Save**.

## Recursos

- Navegação sem página em branco, inclusive com voltar/avançar do navegador.
- Tema claro e escuro.
- Menu para celular.
- Consulta de CEP pela ViaCEP.
- Cadastro e remoção de moradores.
- Cadastro, busca, filtro, pagamento e remoção de contas.
- Totais automáticos e dados salvos no `localStorage`.
